Temas

  • No se puede conectar a la base de datos
  • Ocurrió un error al intentar guardar los cambios
  • Otros problemas

Al iniciar la aplicación se muestra el mensaje "No se epuede conectar con el servidor, vuelva a intentarlo más tarde"

  • Compruebe su conexión a internet

    ¿Puede navegar a su página web favorita?, si no es así lo más probable es que no tenga conexión a internet

  • Tengo conexión a internet pero el problema continua

    Lo más probable es que Java no tenga los permisos para conectarse a internet. Abra el Panel de Control de Java, vaya a General > Configuración de Red, Y de los permisos a Java para que se comunique con el tipo de red a la que está conectado. cierre y vuelva a abrir la aplicación.

Se muestra el mensaje "Ocurrió un error al guardar los cambios"

Verifique que no este intentando guardar texto demaciado grande o imágenes mayores a 1024 KB.
